# MemaxLabs Brand Assets

Official public brand assets for **MemaxLabs** — logos, icons, and brand guidelines served via [GitHub Pages](https://memaxlabs.github.io/public-brand-assets/).

---

## 📂 Repository Structure

```
public-brand-assets/
├── logos/               # Primary logos (SVG and PNG)
│   ├── memax-logo.svg
│   ├── memax-logo-dark.svg
│   ├── memax-logo-light.svg
│   ├── memax-logo.png
│   ├── memax-logo-dark.png
│   └── memax-logo-light.png
├── icons/               # App icons and favicons
│   ├── memax-icon.svg
│   ├── memax-icon.png
│   ├── favicon-32x32.png
│   └── favicon-16x16.png
├── colors/              # Brand color swatches and references
│   └── brand-colors.svg
├── index.html           # GitHub Pages brand portal
├── TERMS.md             # Brand usage terms
├── COPYRIGHT            # Copyright notice
└── README.md            # This file
```

## 🎨 Brand Assets

### Logos

| File | Format | Description |
|------|--------|-------------|
| `logos/memax-logo.svg` | SVG | Primary logo (default, for light backgrounds) |
| `logos/memax-logo-dark.svg` | SVG | Dark variant (for dark backgrounds) |
| `logos/memax-logo-light.svg` | SVG | Light/white variant |
| `logos/memax-logo.png` | PNG (512×512) | Primary logo raster |
| `logos/memax-logo-dark.png` | PNG (512×512) | Dark variant raster |
| `logos/memax-logo-light.png` | PNG (512×512) | Light variant raster |

### Icons

| File | Format | Description |
|------|--------|-------------|
| `icons/memax-icon.svg` | SVG | App icon / logomark |
| `icons/memax-icon.png` | PNG (256×256) | App icon raster |
| `icons/favicon-32x32.png` | PNG (32×32) | Favicon |
| `icons/favicon-16x16.png` | PNG (16×16) | Favicon (small) |

### Brand Colors

| Name | Hex | Usage |
|------|-----|-------|
| Primary | `#0066FF` | Primary brand color |
| Primary Dark | `#0047B3` | Hover / active states |
| Accent | `#00D4FF` | Highlights, accents |
| Dark | `#0A0A1A` | Dark backgrounds, text on light |
| Light | `#F5F7FF` | Light backgrounds |
| White | `#FFFFFF` | White text, light logos |
